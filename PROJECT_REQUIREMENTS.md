# SnapQuest Daily - Требования к проекту

## Основная идея
Фото-челленджи и квесты - игровое соц-приложение, которое превращает создание фотографий в игру на каждый день.

## Ключевые функции

### 1. Daily Challenge (Ежедневный челлендж)
- [ ] Ежедневное задание одинаковое для всех
- [ ] Пользователи выполняют, выкладывают в ленту дня
- [ ] Смотрят чужие интерпретации
- [ ] На следующий день новое слово/тема
- [ ] **AI верификация** - проверка что фото соответствует заданию

### 2. Scavenger Hunt (Фото-охота)
- [ ] Список из N задач на определённый период
- [ ] Отмечать выполненные задания
- [ ] Получать баллы и достижения
- [ ] Прогресс-бар выполнения

### 3. Социальные челленджи (Приватные события)
- [ ] Создание приватных игр для компаний
- [ ] Вход по коду
- [ ] Общая альбом-лента в реальном времени
- [ ] Для свадеб, тимбилдингов, вечеринок

### 4. Профиль пользователя
- [ ] Реальные данные из БД (не mock!)
- [ ] Редактирование профиля (имя, username, аватар)
- [ ] Статистика: фото, стрик, бейджи, XP, уровень
- [ ] История выполненных челленджей
- [ ] Достижения и бейджи

### 5. Загрузка и редактирование фото
- [ ] Камера или выбор из галереи
- [ ] Фильтры (работающие!)
- [ ] Обрезка
- [ ] AI верификация перед отправкой
- [ ] Отзыв о результате (зачтено/не зачтено)

### 6. Геймификация
- [ ] Очки XP за выполнение
- [ ] Уровни
- [ ] Бейджи и достижения
- [ ] Стрик (серия дней)
- [ ] Лидерборд (опционально)
